# Companion Matcher

## Install deps

cd server && npm i
cd ../client && npm i

## Run the client and server on two terminals:

### Terminal 1

cd server && npm run dev

### Terminal 2

cd client && npm run dev

The React app will hot‑reload on localhost:5173, calling the API on localhost:4000.
